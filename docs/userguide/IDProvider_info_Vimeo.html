Synopsis


<?php

    # start a new PHP session

    session_start();



    # include HybridAuth main file

    include "path/to/hybridauth/hybridauth.php";




    # create a new Hybrid_Auth instance 

    $hybridauth = new Hybrid_Auth();

        

    # setup Vimeo adapter 


    $provider_adapter = $hybridauth->setup( "Vimeo" );



    # start login

    $provider_adapter->login();

?>

Registering application

Create a new application at http://www.vimeo.com/api/applications, copy the created credentials (key and secret), put them on HybridAuth config file,

Set http://mywebsite.com/path_to_hybridauth/?hauth.done=Vimeo as Application Callback URL.
