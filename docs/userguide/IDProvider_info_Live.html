HybridAuth, Open Source Social-Sign-On Solution


A PHP Library for authentication through Facebook, Twitter, Google, Yahoo, MySpace, LinkedIn, FourSquare, OpenID and many other Identity providers

HybridAuth User guide » Supported identity providers » Windows Live

Synopsis


<?php

    # start a new PHP session

    session_start();



    # include HybridAuth main file

    include "path/to/hybridauth/hybridauth.php";




    # create a new Hybrid_Auth instance 

    $hybridauth = new Hybrid_Auth();

        

    # setup "Live" adapter 


    $provider_adapter = $hybridauth->setup( "Live" );



    # start login

    $provider_adapter->login();

?>

Registering application

After registing an new application at https://manage.dev.live.com/ApplicationOverview.aspx, copy the created credentials (key and Secret), put them on HybridAuth config file,

Make sure to put your correct website adress in the "Redirect Domain" field.

 Description Open Authorization Web Resource Authorization Protocol (OAuth WRAP) is an implementation of OAuth that enables users to provide consent to applications. With this consent, applications can access user information that is stored in protected resources, such as a Windows Live service.
