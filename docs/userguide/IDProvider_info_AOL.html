HybridAuth, Open Source Social-Sign-On Solution


A PHP Library for authentication through Facebook, Twitter, Google, Yahoo, MySpace, LinkedIn, FourSquare, OpenID and many other Identity providers

HybridAuth User guide » Supported identity providers » AOL

Notice: AOL adapter temporary has been removed from hybridauth. Their API seems to be broken. Use OpenID for AOL instead: http://openid.aol.com/screenname

Synopsis

session_start();

include "path/to/hybridauth/hybridauth.php";

$hauth = Hybrid_Auth::setup( "AOL" );

$hauth->login();

Registering application

Go to http://dev.aol.com/keys to create a new application. Fill required input, then copy the created application credentials (DEVID) and set it on HybridAuth config file,
