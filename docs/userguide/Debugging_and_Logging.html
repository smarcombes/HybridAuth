HybridAuth, Open Source Social-Sign-On Solution


A PHP Library for authentication through Facebook, Twitter, Google, Yahoo, MySpace, LinkedIn, FourSquare, OpenID and many other Identity providers

HybridAuth User guide » Debugging and Logging

To enable Logging, you have to set $GLOBAL_HYBRID_AUTH_DEBUG_MODE to TRUE on hybridauth/hybridauth.php configuration file.

By default the log file can be located in the HybridAuth temporary folder. The complete path to log file on your server should be something like this: /www/mywebsite.com/temp/log.log

There is 3 levels defined in HybridAuth Logger :

      INFO : Log a message at the info level.
      DEBUG : Log a message at the debug level.
      EURROR : Log a message at the error level.
