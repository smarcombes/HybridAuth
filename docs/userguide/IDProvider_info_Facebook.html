HybridAuth, Open Source Social-Sign-On Solution


A PHP Library for authentication through Facebook, Twitter, Google, Yahoo, MySpace, LinkedIn, FourSquare, OpenID and many other Identity providers

HybridAuth User guide » Supported identity providers » Facebook

Synopsis


<?php

    # start a new PHP session

    session_start();



    # include HybridAuth main file

    include "path/to/hybridauth/hybridauth.php";




    # create a new Hybrid_Auth instance 

    $hybridauth = new Hybrid_Auth();

        

    # setup Facebook adapter 


    $provider_adapter = $hybridauth->setup( "Facebook" );



    # start login

    $provider_adapter->login();

?>

Registering application

Go to https://www.facebook.com/developers/apps.php to create a new application. Fill required input, then copy your created application credentials (App ID and App Secret) and set them on HybridAuth config file,
